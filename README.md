The **Warehouse Logistics & Order Fulfillment System** is a C++ application built using the **Win32 GUI** that simulates real-world warehouse operations. The system manages categories, items, inventory stock, and customer orders while demonstrating the practical use of fundamental data structures.

This project was developed from scratch by a **team of four** as an academic and learning-focused implementation of warehouse logistics.
